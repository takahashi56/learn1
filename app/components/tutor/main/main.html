Import Employee

Rather than add employees one by one you can import employees from a CSV formatted file.

The file should be a text file with following columns each seperated with a commas:

First Name,Last Name,DOB,Mobile,Username,Password

Example text file format:

John,Smith,20/12/1970,07882770123,jsmith,Pass88
George,Jones,18/06/1985,0312740123,gjones,Pass12
Harry,Flow,30/10/1959,0771551123,hflow,Pass44
